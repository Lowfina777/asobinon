**マルチプレイのチャット**を、**もっと楽しく**したいですよね。  
**文字に色を付けたり**、**太字にしたり**、**ニコちゃんマークなどの記号**を付けたり…  
今回紹介するのは、NukeDuckさん制作の**「Color Code GUI」**です。  
Bukkitサーバーでのチャットで、****文字の装飾やUnicode記号の入力**が簡単になります。**
