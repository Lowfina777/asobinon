# Firebaseを用意する

ナポアンのマイクラNEXTでは、これには以下が含まれます。

- **Firestore**
  - 評価・コメントを記録する
  - ユーザーのガチャデータを記録する
- **Cloud Functions for Firebase**
  - 記事を取得・ツイート数を取得する
  - エンティティガチャを回す
  - 定期的にFirestoreをバックアップする

それでは、FIirebaseを用意しましょう。
