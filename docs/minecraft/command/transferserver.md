# transferserver (削除済み)

| 項目 | 内容 |
| --- | --- |
| コマンド | `transferserver` |
| 目的 | サーバーに接続する |
| 必要な権限レベル | 0 |
| コマブロで実行 | 不可 |
| 適用対象 | - |

:::warning

**1.2.3で削除された。記録として残しておく。**

:::

## 構文

[[dt|サーバーに接続する]]
|```
|/transferserver <サーバー> <ポート>
|```

## 引数

[[dl| <サーバー> (必須)]]
|接続するサーバーのアドレスを指定する。数字のアドレスの場合はダブルクオーテーションで囲まなければならない。
|例: `"123.456.78.9"`

[[dl| <ポート> (必須)]]
|サーバーのポート番号を指定する

## 実行結果

次の場合は失敗する:

- インターネット接続がなかった
- アドレスが間違っていた
- サーバーが存在しなかった
- サーバーがオフラインだった

成功すると、現在のサーバーから切断し、指定したサーバーに参加する。
