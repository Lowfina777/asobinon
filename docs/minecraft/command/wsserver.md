# wsserver

| 項目 | 内容 |
| --- | --- |
| コマンド | `wsserver` |
| 目的 | WebSocketサーバーに接続する |
| 必要な権限レベル | 2 (教育版では0) |
| コマブロで実行 | 不可 |
| 適用対象 | - |

## 構文

[[dl|WebSocketサーバーに接続する]]
|```
|/wsserver <サーバーURI>
|```

## 引数

[[dl| <サーバーURI> (必須)]]
|接続するWebSocketサーバーのURIを指定する。

## 実行結果

WebSocketサーバーに接続できなかった場合は失敗する。

なお、**このコマンドは主に教育版で使われる**ものであり、生徒の端末からサーバーに接続するといった使い方がされる。
