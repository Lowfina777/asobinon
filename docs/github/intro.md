# About this section

レポジトリのファイルごとの解説を置いています。

:::warning

**ドットから始まるファイルに対応したページを作るには、ゼロ幅スペースを使ったmdファイルを作らないといけません。**

```
​.hogehoge.md
```

▲これをコピペしてファイル名にしてください。**ゼロ幅スペースがないと、隠しファイル扱いになり、ページが生成されません。**

:::
