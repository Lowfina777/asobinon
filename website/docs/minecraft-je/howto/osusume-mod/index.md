**Java版のおすすめMODのまとめです。**定番の軽量化MOD「Optifine」や、普段は見れない情報が見える便利なMOD、ミニマップMOD、細かい部分を改善してくれるMOD… それぞれのMODの導入方法も一緒にまとめています。

Optifineなどシステムに深く干渉するMODは、必ず最後の桁までバージョンと合致したものを入れてください。
