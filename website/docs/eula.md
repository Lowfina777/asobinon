この利用規約（以下，「本規約」といいます。）は、<https://asobinon.org> 及び <https://client.asobinon.org> で公開されている「アソビノン」（以下，「当サイト」といいます。）がウェブサイト上で提供するサービス（以下，「本サービス」といいます。）の利用条件を定めるものです。登録ユーザーの皆さま（以下，「ユーザー」といいます。）には，本規約に従って，本サービスをご利用いただきます。

## 1（適用)

1. 本規約は，ユーザーと当プロジェクトとの間の本サービスの利用に関わる一切の関係に適用されるものとします。
2. 当プロジェクトは本サービスに関し，本規約のほか，ご利用にあたってのルール等，各種の定め（以下，「個別規定」といいます。）をすることがあります。これら個別規定はその名称のいかんに関わらず，本規約の一部を構成するものとします。
3. 本規約の規定が前条の個別規定の規定と矛盾する場合には，個別規定において特段の定めなき限り，個別規定の規定が優先されるものとします。
4. 以下、「レポジトリ」は[GitHub上のレポジトリ](https://github.com/aelyone/asobinon)を指します。
5. 以下、「GitHubを使用」とは、以下の行為を指します。
    1. forkしたレポジトリからプルリクエストをすることで編集を提案する
    2. レポジトリのメンバーとして、レポジトリ上のファイルを直接編集する
    3. レポジトリのメンバーとして、ローカルから直接変更をプッシュする
    4. レポジトリにissueを作成する
    5. レポジトリのissueにコメントを書き込む

## 2（利用登録)

1. 本サービスでは、ユーザーが自ら希望した際に、GitHubを経由してログインすることができます。
2. GitHubから取得するデータには、以下が含まれます。プログラムの仕様変更により、**ここにないデータが送信されている場合もあります。**

   1. メールアドレス
   2. ユーザーID(数字・英数字両方)
   3. アバターの画像URL
3. コンテンツの編集にGitHubを使用するため、レポジトリの編集に関わったGitHubアカウントを、1ユーザーとして扱うものとします。
4. GitHubを介さない編集は現在のところ受け付けておりません。

### 2-1.（画像アップローダー)

1. 画像アップローダーを、編集補助を目的として設置しています。
2. 画像アップローダーの利用は、GitHubでのログインを必須とします。
3. GitHub組織への参加は必要ありません。
4. アップロードする画像の権利に関するトラブルは、アップロードしたユーザーが対処するものとします。
   1. GitHubのアカウントからメールを通じた連絡をします。
5. 画像のアップロード後に、自動で画像のデータ解析をし、不適切な画像を検出します。
   1. NSFW判定は**Google Cloud Vision API**の判断を基準とします。
      1. APIによる「Racy」「Adult」の数値を独自の計算式で換算し、画像に紐づけます。
      2. 換算結果は`0,1,2,3`の4段階になり、`2,3`は記事への貼り付けを許可しません。
      3. 以上の基準により、ゲームのスクリーンショットは、ほとんど`0`に分類されることを想定しています。
      4. ゲーム攻略という性質上「水着キャラ」が掲載されるかもしれませんから、`1`は暫定処置として許可しています。
   2. **AIにNSFW判定された画像もURLが発行されていますが、運営以外は入手できません。**
   3. **運営が予告なく画像を削除する場合があります。**運営が削除した画像の復元は申し受けられません。
   4. NSFW判定に意義がある場合は、運営に再審議を申し込むことができます。その際はメールを返信に使います。
   5. 再審議の結果、適切だと判断された場合は、メールで連絡します。
6. ユーザーごとに画像は分類されており、運営は誰がアップロードしたか追跡できます。
   1. ユーザーごとに以下のゲータが全て保存されています。
      1. 画像のアップロード枚数
      2. NSFW判定の回数
      3. レベルの累積
   2. NSFW率が高いユーザーは、個別に警告するだけでなく、利用停止措置を撮る場合があります。**組織に参加している場合は、役職に関係なく厳正に処罰します。**
7. 記事に貼っているか否かに関係なく、画像はデータベース上に保管されます。
8. 画像をデータベース上から削除する問い合わせは、**URLを連絡**していただく必要があります。
   1. その際の記事の修正はベストエフォートとして提供し、即時修正はしません。
9. アカウントの削除をしても、要求がない限り画像は削除しません。

### 2-2（編集補助ページ）

1. 編集ボタンを押した後、画像アップローダがスムーズに使えるよう、「編集補助ページ」が用意されています。
2. 編集補助ページはログインなしで使えます。
3. しかし、GitHubを使うため、ページ移動先でGitHubアカウントが必要になります。
